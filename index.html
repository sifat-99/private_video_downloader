How to get the source code

  • 1
    Open the Facebook video in a new tab. Make sure you can see it playing or loaded.
  • 2
    Press Ctrl + U (Windows/Linux) or Option + Cmd + U (Mac) to open the page source. Alternatively, right-click and click View Page Source.
  • 3
    Select everything by pressing Ctrl + A (Windows/Linux) or Cmd + A (Mac) and copy it (Ctrl + C / Cmd + C).
  • 1
    Open the Facebook video in Firefox.
  • 2
    Right-click anywhere on the page and select View Page Source, or press Ctrl + U / Cmd + U.
  • 3
    Copy all content (Ctrl + A then Ctrl + C).
  • 1
    Open Safari, go to Settings > Advanced, and make sure "Show Develop menu in menu bar" is enabled.
  • 2
    Open the Facebook video, right-click the page, and select Show Page Source, or press Option + Cmd + U.
  • 3
    Click inside the source panel, select all, and copy the code.

How to save the video

If clicking "Download" simply opens the video in a new tab instead of saving it, please try the following:

  • Right-click the Download button and choose "Save Link As...".
  • On mobile devices, press and hold the Download button, then select "Download Link".
  • If the video opens in a new tab, click the three dots vertical menu on the video player and choose Download, or press Ctrl + S / Cmd + S to save.
